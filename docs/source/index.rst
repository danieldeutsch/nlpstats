NLP Stats
=========
This package contains statistical tools for NLP.
Thus far, it provides functions for:

- :ref:`meta-evaluating text generation metrics <nlpstats.correlations>`

The library can be installed via pip:

.. code::

    pip install nlpstats

The source code can be found on `GitHub <https://github.com/danieldeutsch/nlpstats>`_.
